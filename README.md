# WebPortfolio - Gewa Andoko

🌐 **Portfolio Website** dengan tema **Aksara Indonesia & Batik**

Sebuah website portofolio pribadi yang menggabungkan keindahan budaya Indonesia (Aksara Jawa & motif Batik) dengan teknologi web modern. Dibuat untuk **Gewa Andoko**, lulusan Sistem Komputer.

## 🚀 Fitur

- **Tema Batik & Aksara** — Desain terinspirasi dari motif batik Indonesia dan aksara Jawa
- **Hero Section** — Animasi ornament lingkaran berputar dengan aksara di tengah
- **Responsive Design** — Tampilan optimal di desktop, tablet, dan mobile
- **Skill Bars Animasi** — Progress bar keahlian dengan animasi scroll
- **Timeline Pendidikan** — Riwayat pendidikan dengan desain timeline interaktif
- **Project Cards** — Grid proyek dengan efek hover
- **Contact Form** — Formulir kontak dengan validasi dan notifikasi
- **Smooth Scrolling** — Navigasi smooth scroll ke setiap section
- **Parallax Effect** — Efek paralaks pada background batik

## 🛠️ Teknologi

- HTML5
- CSS3 (Custom Properties, Grid, Flexbox, Animations)
- JavaScript (Vanilla JS)
- Font Awesome 6
- Google Fonts (Playfair Display & Inter)
